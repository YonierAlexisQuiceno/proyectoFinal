📝 Reporte para Descargar
📚 ¿Cómo funciona la descarga de archivos?
  1. Obtenemos el texto con document.getElementById()
  2. Creamos un new Blob([texto]) — convierte texto en datos binarios
  3. Generamos una URL temporal con URL.createObjectURL(blob)
  4. Creamos un enlace <a> con atributo download
  5. Simulamos clic con enlace.click() → ¡se descarga!
  6. Limpiamos memoria con URL.revokeObjectURL()
